5ï¸âƒ£ Conditions in Python (if, else, elif)

Conditions ka use decision lene ke liye hota hai. Matlab agar koi condition true hai toh ek kaam hoga, warna dusra kaam hoga.

ğŸ”¹ if Statement

if tab use hota hai jab sirf ek condition check karni ho.

Agar age 18 ya usse zyada hogi, tab message print hoga.

ğŸ”¹ if - else Statement

Jab condition true ho tab ek kaam, aur false ho tab dusra kaam karna ho.

ğŸ”¹ if - elif - else

Jab multiple conditions check karni ho tab elif use hota hai.

    print("Grade A")

    print("Grade B")
else:
    print("Grade C")

Python upar se neeche condition check karta hai. Jo pehli true hogi wahi execute hogi.

6ï¸âƒ£ Loops in Python

Loop ka use tab hota hai jab hume kisi kaam ko baar-baar repeat karna ho.

ğŸ”¹ for Loop

for loop ka use tab hota hai jab hume pehle se pata ho kitni baar loop chalega.

ğŸ”¹ while Loop

while loop tab use hota hai jab tak condition true rahe.

Jab i ki value 5 se zyada ho jayegi, loop ruk jayega.

âš  Infinite Loop (Galti se)

Agar condition kabhi false na ho, toh loop infinite chal sakta hai.

    print(i)

âš  Isme i = i + 1 nahi hai, isliye loop kabhi khatam nahi hoga.

ğŸ”¹ break Statement

break loop ko beech me hi rok deta hai.

    if i == 5:
        break
    print(i)

ğŸ”¹ continue Statement

continue current iteration ko skip karta hai.

    if i == 3:
        continue
    print(i)

âœ… Summary

  • for loop â†’ fixed times ke liye
  • while loop â†’ condition ke according
  • break â†’ loop stop
  • continue â†’ iteration skip

Function ek aisa block hota hai jisme hum code ko ek baar likh kar baar-baar use kar sakte hain.
