What is Python?

Python is a simple and powerful programming language. It is used in web development, data science, AI, automation, and many more fields. Python is famous because it is easy to read and write.

How to Install Python

  1. Go to python.org
  2. Download latest Python version
  3. Install and tick Add Python to PATH
  4. Open Command Prompt and type python

Python Basic Syntax

Python code is very simple compared to other languages.

print("Hello, World!")

No semicolon, no brackets – simple and clean.

Variables in Python

Variables are used to store data.

name = "Gaurav"
age = 18
print(name)
print(age)

Data Types

  • int – Numbers (10, 20)
  • float – Decimal numbers (10.5)
  • str – Text ("Python")
  • bool – True / False
x = 10
y = 10.5
z = "Python"

Loops in Python

Loops are used to repeat code.

For Loop

for i in range(5):
    print(i)

While Loop

i = 1
while i <= 5:
    print(i)
    i += 1

Functions

Functions are used to reuse code.

def greet():
    print("Hello Python")

greet()
